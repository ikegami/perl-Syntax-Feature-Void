Syntax-Feature-Void

Syntax::Feature::Void is a lexically-scoped pragma that
provides the void operator to force void context.
Not that anyone needs that.


INSTALLATION

To install this module, run the following commands:

   perl Build.PL
   ./Build
   ./Build test
   ./Build install


DEPENDENCIES

This module requires these other modules and libraries:

    Perl 5.14.0
    Module::Build 0.36_14       (For installation only)
    Software::License::CC0_1_0  (For installation only)
    Test::More                  (For testing only)
    Devel::CallParser
    Lexical::Sub
    strict
    syntax 0.004
    version
    XSLoader
    warnings
